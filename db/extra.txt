int sum_of_digits(int n) をできるだけ、{} や変数を使わずに書け。

n * m と同じ計算をする関数 int times(int n, int m) を定義せよ。times 中で * を使うのは反則とする。

++ のみを使い、x + y と同じ計算をする関数 int add(int x, int y) を定義せよ。

+ や * を使わず、n * 2 と同じ計算をする関数 int doubles(int n) を定義せよ。
